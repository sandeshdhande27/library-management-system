# 📚 Library Management System

This is a backend project built using Spring Boot for managing a library system.

## Features
- User registration and login
- JWT authentication
- Add, update and delete books
- Issue and return books
- Membership validation
- User history tracking
- Category-based reporting

## Tech Stack
- Java
- Spring Boot
- SQL Server
- Hibernate

## How to Run
1. Clone the repo
2. Run SQL script from `database/schema.sql`
3. Update DB config in `application.properties`
4. Run application

## APIs
